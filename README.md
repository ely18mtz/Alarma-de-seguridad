# Alarma-de-seguridad
Prototipo: Alarma de seguridad para vivienda
El objetivo es  implementar un sistema de autenticación mediante contraseña garantiza que únicamente las personas autorizadas puedan ingresar. Este mecanismo tiene como objetivo reducir el riesgo  de intrusiones o accesos no deseados.
Se ha implementado el uso de la Nexys 4, la cual es una placa de desarrollo para FPGAs (Field-Programmable Gate Array). Está diseñada para aplicaciones educativas y utiliza una FPGA de la familia  Xilinx Artix-7.
El lenguaje de programacion es en verilog (Vivado), el cual se eproporciona en este arcrchivo.
